# open-api-validator
Open API Validator to validates request and response against Swagger 2 and Open API contracts 

```bash
Usage: oas-validator [-d=<requestBody>] -s=<schema> [-X=<method>]
                     [-H=<headers>]... <uri>
      <uri>
  -d, --data=<requestBody>
  -H, --headers=<headers>
  -s, --schema=<schema>
  -X, --method=<method>
```
